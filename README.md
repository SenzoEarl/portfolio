# Senzo Earl Maseko | Full-Stack Portfolio

Professional, recruiter-ready portfolio website built with React, Vite, and Tailwind CSS. Featuring high-performance animations, mobile-first design, and deep technical integration.

## 🚀 Features

- **Performance-First Architecture**: Fast initial load with optimized assets.
- **Recruiter-Focused UI**: Minimalist design highlighting skills, experience, and projects.
- **Mobile Responsive**: Fully adaptive layout for all screen sizes.
- **Animated Interactions**: Smooth transitions using Framer Motion.
- **SEO Optimized**: Semantic HTML, Open Graph tags, and JSON-LD structured data.
- **Project Highlighting**: Detailed project cards with tech stack and major achievements.

## 🛠 Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Deployment**: Optimized for Vercel/Netlify/Vite-ready environments.

## 📂 Project Structure

```bash
/src
  /components     # Modular React components (Hero, Navbar, etc.)
  /data           # Normalized portfolio content data
  /types          # TypeScript interfaces for type safety
  App.tsx         # Main application assembly
  main.tsx        # Entry point
  index.css       # Global styles and Tailwind configuration
```

## 🏗 Setup & Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## 📈 Performance Considerations

- **Asset Optimization**: Uses system fonts and minimal external dependencies.
- **Lazy Loading**: Images and components enter the viewport with entry animations.
- **Static Assets**: Ready for static hosting with maximum cacheability.
